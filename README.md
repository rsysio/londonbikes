## Prerequisites
- GNU Make
- docker
- Set App Id and API Key

### API keys
```
export TFL_APP_ID=<your App ID>
export TFL_API_KEY=<your Key>
```

###Â Usage with docker
```
make build
make help
```

### Install without docker

Install Python 3.8 or later
```
make install
```

run:
```
./londonbikes help
```
